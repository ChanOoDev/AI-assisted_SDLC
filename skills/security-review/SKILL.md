# /skills/security-review/SKILL.md

## Objective

Review implementation against OWASP and enterprise security rules.

## Verify

- JWT validation
- RBAC enforcement
- input validation
- secret management
- secure logging
- audit logging
- secure error handling

## High Risks

- auth bypass
- sensitive logs
- SQL injection
- insecure token storage
- exposed stack traces
