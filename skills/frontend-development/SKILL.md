```md
# /skills/frontend-development/SKILL.md

## Objective
Implement enterprise-grade React TypeScript frontend.

## Stack
- React
- TypeScript
- Vite
- React Query

## Required Rules
- Feature-based structure
- No business logic in UI
- Typed API models
- Handle loading/error states
- RBAC-aware UI
- Reusable validation

## Workflow
1. Read UX spec
2. Read API spec
3. Create typed models
4. Create API client
5. Create feature components
6. Add validation
7. Add tests

## Forbidden
- direct fetch in components
- untyped API responses
- insecure token storage
- giant components
```
