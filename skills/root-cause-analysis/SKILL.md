```md
# /skills/root-cause-analysis/SKILL.md

## Objective
Perform structured production issue RCA.

## RCA Flow
1. Identify symptoms
2. Gather logs/traces
3. Identify impacted systems
4. Build timeline
5. Identify root cause
6. Identify contributing factors
7. Define corrective actions

## Verify
- retry behavior
- deployment changes
- infra changes
- DB impact
- integration failures
- concurrency issues

## Output
- root cause
- impact
- recovery action
- prevention action
```
